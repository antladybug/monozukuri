MESH SDKをつかう

MESHのSDK(Software Development Kit)の公式サイトでサインインして、オリジナルのブロックを作成します。
SDK公式サイト
次のような画面にて、
下の赤枠の「MESH SDKをつかう」ボタンをクリックします。
次のような画面に遷移します。
赤枠のように、ブロックの説明を入力（内容は任意）し、「Save」ボタンをクリックします。
次の赤枠の「Code」ボタンをクリックします。
下のような画面に遷移します。青枠の領域にプログラムを入力します。言語はJavaScriptです。
log(" こんにちは！ "); return { resultType : "continue" };
