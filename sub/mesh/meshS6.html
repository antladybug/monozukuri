ログ出力するSDKをつくる

MESHのSDK(Software Development Kit)の公式サイトでサインインして、オリジナルのブロックを作成します。
SDK公式サイト
次の画面にて、操作します。
下の赤枠の「MESH SDKをつかう」ボタンをクリックします。
次のような画面に遷移します。
赤枠のように、ブロックの説明を入力（内容は任意）し、「Save」ボタンをクリックします。
赤枠の①New Functionの▶をクリックしてメニューを広げ、赤枠の②Code▶をクリックします。
下のようにcodeの入力欄が開きます。青枠の領域にプログラムを入力します。言語はJavaScriptです。
codeには、『Initialize』『Receive』『Execute』『Result』のメソッドが、タブで分かれています。
出典元:https://meshprj.com/sdk/doc/ja/
詳細は、次のマニュアルを解読しましょう。
リファレンスページへ
『Execute』メソッドに、次のコードを入力しましょう。ログに文字列『こんにちは』を出力する命令です。
Execute
log(" こんにちは！ ");
return {
 resultType : "continue"
};
スクロールアップし、赤枠「Save」で、保存します。
これで、SDKの作成は完了です。
ここからは登録したSDKを、MESHブロックに連携します。MESHアカウントでログインした上で、操作してください。
MESHアプリのインストール先フォルダの赤枠『MESH.exe』を、クリックして起動します。
MESHアプリの右側「ブロック」のエリアを下にスクロールダウンします。
カスタム枠内の、赤枠『追加』を、クリックします。
次の取得中ウィンドウが、終わるのを待ちます。
次のようなウィンドウに遷移します。
登録した赤枠『New Block』を、クリックします。
赤枠『追加』を、クリックします。
次のウィンドウに遷移したら、赤枠『OK』を、クリックします。
MESHアプリの右ブロックに赤枠『New Block』が追加されたことを確認します。
赤枠「①New Block」を選択し、「②キャンパス領域」へドラッグアンドドロップします。
任意のブロック青枠「ブロック」と赤枠「New Block」を連携して、動作確認しましょう。
イベント発生をトリガ(ここでは青ブロックのボタンを1回クリック)に、MESHアプリのログに『こんにちは』の文字列が送信されたら、成功です。
[ログの確認方法]
MESHアプリの右下「カスタム」グループ内の『ログ』をクリックします。
イベント発生後に、下のログ領域を確認しましょう。
以上です。
