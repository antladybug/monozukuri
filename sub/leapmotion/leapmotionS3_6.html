ピースサインを検知する

次の内容のプログラムを作成してください。
[要件]
ピースサインを検知したら、対象の指（人差し指と中指）を「特定の色（赤と白い以外、任意）」で表示するプログラム
ピースサインの定義：人差し指と中指以外の指が手の中に隠れている状態
※前回の演習問題の回答例です。参考にしてください。
[要件]
手のひらで隠れていない指を、『青色』で表示する。
test_finger.pde
void drawFingerTip(Hand hand) {
 FingerList fingers = hand.fingers();
 for(int i = 0; i < fingers.count(); i++) {
  Finger finger = fingers.get(i);
  Vector tipPos = finger.tipPosition();
  Vector tipPosNorm = iBox.normalizePoint(tipPos, false);
  if(finger.isExtended() == true) {
   fill(0, 0, 255);
  } else {
   fill(255, 255, 255);
  }
  pushMatrix();
   translate(tipPosNorm.getX() * width,
   (1 - tipPosNorm.getY()) * height,tipPosNorm.getZ() * 50);
   noStroke();
   lights();
   sphere(20);
  popMatrix();
 }
}
45行目の青字の箇所を変更します。
ここは検知した指のオブジェクトをループ処理で1本ずつ、色描画している箇所です。
44行目の『isExtended』は、対象の指が立っている（隠れていない）かどうかを返す関数です。
判定処理で、fill関数をコールします。
この関数は、引数が3つとも0の場合は白と描画し、赤は第1引数のみ255で指定します。
青は、第3引数のみ255と指定するため、上記のようなコード修正で対応可能です。
以上です。
